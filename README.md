# alefesouza

> The [Alefe Souza](https://alefesouza.com) CLI

![](screenshot.png)

## Usage

To install the CLI, first install Node.js, then:

```bash
$ npx alefesouza
```

and run:

```bash
$ alefesouza
```

To install on Node projects, run:

```bash
$ npm install alefesouza
```

and

```js
console.log(require('alefesouza'));
```

To get my contact information.
